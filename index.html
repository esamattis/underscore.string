STRING

Underscore.string is a string manipulation extension for JavaScript that you can use with or without Underscore.js [npm install underscore.string]

Description

Underscore.string is a JavaScript library for comfortable manipulation with strings, extension for Underscore.js inspired by Prototype.js, Right.js, Underscore and beautiful Ruby language.

Underscore.string provides several useful functions: capitalize, clean, includes, count, escapeHTML, unescapeHTML, insert, splice, startsWith, endsWith, titleize, trim, truncate and so on.

  • Works well with Underscore 1.3.3
  • If you found a bug open an issue on Github Issues
